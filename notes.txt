oru oru project ku etha code da use pannanu.

npm init = one by one na pogurathuku
npm init -y = kudutha pothu athuvey pathuku. (json file) create pannum.
npm i express = expres nu oru package ja instal panna
node index.js = file la runn panna.
npm i -g nodemon = project ta live la run panna.


const express = require('express');     downlod pannunathu file la use pannurathuku etha line in js file.
const app = express();                  express sa oru function na call pannuro.
const port = 3000;                      output etha port la run aganunu namatha mudiivu pannanu.namba system la same port number repect aga kudathu.
app.listen()                            full code dayum oru port la run panna.

app.listen(port,()=>{
    console.log("Server is running...",port);
})                                              server la run aguthanu run panna.
